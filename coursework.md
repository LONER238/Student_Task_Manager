## 1. Introduction

This coursework folder contains the supporting material for my Software Development Skills Full-Stack project. The main project is a full-stack web application called **Student Task Manager**.

The purpose of the project is to help students manage their study tasks more easily. Users can create tasks, view task lists, update tasks, and delete tasks. The project was built using the MERN stack.

---

## 2. Technologies Used

The project uses the following technologies:

- **MongoDB** – database for storing tasks
- **Express.js** – backend framework
- **React** – frontend user interface
- **Node.js** – backend runtime environment
- **Vite** – frontend development tool
- **JavaScript** – main programming language
- **HTML and CSS** – structure and styling
- **Git and GitHub** – version control and project submission
- **VS Code** – development environment
