## Project Description

Student Task Manager is a simple MERN-stack web application created for the Full-Stack module.

The purpose of the application is to help students manage their study tasks. The user can add new tasks, view existing tasks, mark tasks as completed, undo completed tasks, and delete tasks.

This project demonstrates how MongoDB, Express.js, React, and Node.js can work together in one fullstack application.
